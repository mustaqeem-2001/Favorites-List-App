# Favorites-List-App
A simple React app to manage a list of favorite items. Showcasing react 

## Features
- Add new items to a list
- Mark items as favorite
- Delete items from the list

Usage
Type an item in the input field and click "Add"
Click the "Favorite" button to mark it
Optional: Delete an item from the list

## Tech Stack
- React (Vite)
- JavaScript
- HTML / CSS

## Installation
1. Clone the repo

git clone https://github.com/yourusername/favorites-list.git
